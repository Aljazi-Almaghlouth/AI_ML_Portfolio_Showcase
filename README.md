# AI Research & Applied Machine Learning Portfolio

A Netflix-style Streamlit portfolio showcasing applied AI and machine learning projects across NLP, LLMs, computer vision, time-series forecasting, and financial risk analytics.

## Projects
- Financial News Classification — Reuters Dataset
- Executive Text Summarization — BBC News Summary Dataset with T5 Fine-Tuning
- Customer Sentiment Intelligence — Yelp Review Polarity with BERT
- LoRA Sentiment Analysis — IMDB Reviews with DistilBERT + PEFT
- Drone Detection — Computer Vision for airport environments
- Nasdaq Time-Series Decomposition & Forecasting
- Market Volatility, VaR & Stress Testing

## Run locally
```bash
pip install -r requirements.txt
streamlit run app.py
```
